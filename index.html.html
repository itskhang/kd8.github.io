Web Architecture

Course Description

Students shall gain a fundamental understanding of how web applications function, the role of client­side and server­side technologies, how web services are designed and operate, and the technologies required to support mobile devices. Students will gain system development experience with current web development tools and platforms.

Prerequisites

CIS 350

Expectations

To be successful you should allow 6­-9 hours per week to study this material. The material is presented via video and reading.
